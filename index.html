This page demonstrates how CSS selectors work.

id

ids can be applied to only one item on a page

Here's how we hook an id attribute to an element:

id="id-div"

Here's how the CSS is applied to the element:

div#id-div{
    background-color:yellow;
}

Inside my-div
class

classes can be applied more than one item on a page

Here's how we hook a class attribute to an element:

class="my-list"

Here's how the CSS is applied to the element:

li.my-list{
    background-color:blue;
}

compound

A compound selector allows multiple selectors separated by commas. In this way this allows us to apply the same rules for several selectors.

Here's how the CSS is applied to the element:

div#new-div,li.new-list{
    background-color:blue;
}

child

A child selector allows us to access the next immediate elements inside the current element.

Here's how the CSS is applied to the element:

ul.my-kids > li{
    background-color:green;
}

Here's what it looks like:

descendant

Using a descendant selector we can acess all elements at any depth within the current element.

Here's how the CSS is applied to the element:

div.grandkids li {
    background-color:red;
}

multiple class

We can apply multiple classes to an element by lacing class names separated by spaces in the class attribute.

Here's how we hook multiple classes to an element:

class="add-border add-background"

Here's how the CSS is applied to the element:

.add-border{
    border:1px solid #000;            
}
.add-background{
    background-color:gold;
}
